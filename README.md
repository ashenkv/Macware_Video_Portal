# SCC_Video_Portal(Macware)
(*Still under development)
A full-stack educational platform where teachers manage courses and students access video lessons securely. Built with React, Node.js, Express, and Prisma ORM, it features JWT authentication, role-based access (teacher/student), and Cloudinary-powered video upload &amp; streaming. eachers can create courses, register students, and upload videos; students only see content from enrolled courses. The backend uses MySQL via Prisma with secure relations, while the frontend provides intuitive dashboards using React components and react-player for smooth playback. Multer handles file uploads, and environment variables are managed with dotenv. Deployed on Vercel (frontend) and Render (backend), this LMS-style app ensures scalable, real-world functionality. Perfect for institutions needing a lightweight, self-hosted video learning system.

Tech Stack: React, Node.js, Express, Prisma, MySQL, Cloudinary, JWT, CORS, Multer, Vercel, Render
Features: User Auth, Course Management, Video Upload & Streaming, Student Enrollment, Role-Based Access
